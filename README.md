## Installing 🚧

Clone this repo:

```

  git clone https://github.com/RareFishes/rarefishes.crypto.git

```

```

  cd rarefishes.crypto

```

## Running 🏃

Development mode:

```

  yarn dev

```

Production mode:

```

  yarn build

```

```

  yarn start

```

## Testing 🏃

Running the tests:

```

  yarn test

```

Running cypress for end2end tests:

```

  yarn test:cypress

```

## Development :gear:

Install husky for git hooks:

```

  npx husky install

```

Start coding...

```

  yarn dev

```
